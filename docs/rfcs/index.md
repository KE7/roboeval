# RFC / ADR Index

This directory contains lightweight architecture decision records for roboeval.

The records are intentionally short.

Each record captures one decision that affects public extension behavior.

Each record uses the same structure:

- Status
- Context
- Decision
- Consequences
- Alternatives Considered

Accepted records describe current project behavior.

They are not a roadmap.

They are not benchmark reports.

They are not a replacement for the user-facing guides.

Use them when you need to understand why a contract exists.

Use [../architecture.md](../architecture.md) for the system overview.

Use [../design.md](../design.md) for design principles.

Use [../extending.md](../extending.md) for implementation steps.
